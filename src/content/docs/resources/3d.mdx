### Stable Point Aware 3D (SPAR3D)
<Badge text="Stability-AI" variant="tip" />

A state-of-the-art open-source model for fast feedforward 3D mesh reconstruction from a single image using point cloud conditioning.
* Features:
  + 3D mesh reconstruction
  + Single image input
  + Point cloud conditioning
  + Open-source
* [Read the Documentation](https://github.com/Stability-AI/stable-point-aware-3d/blob/main/README.md#comfyui-extension)

### Hyperfy Hitbox
<Badge text="Blender Add-on" variant="tip" />

A blender add-on that allows you to create a nested Empty - cube, and collider. You can also select a pre-existing object and generate a collider for it with a button.
* Features:
  + Nested Empty creation
  + Cube and collider creation
  + Generate collider for existing objects
* [Visit Hyperfy Hitbox on GitHub](https://github.com/scoegrams/hyperfyhitbox)
