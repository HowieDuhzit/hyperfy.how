### Prerequisites

**Node 22.11.0+ (eg via [nvm](https://github.com/nvm-sh/nvm/blob/master/README.md))**

### Install

```sh frame="none"

git clone https://github.com/hyperfy-xyz/hyperfy.git my-world
cd my-world
cp .env.example .env
npm install
npm run dev

```
